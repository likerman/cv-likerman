# CV summary for the UNESCO IGCP call (Latin America & Caribbean)

Three-page English CV summary tailored to UNESCO's "Call for Expressions of
Interest: Geoscience Funding for Latin America and the Caribbean" (IGCP,
deadline 13 Jul 2026). Uses the same Awesome-CV design as the other CVs in
this repository. Content is drawn from the full CV (`CV_JSPS_English/full_cv.tex`,
the most up-to-date version) and curated/reordered around the call's stated
priorities: IGCP themes (Geodynamics, Earth Resources), capacity-building,
and regional (Latin America) cooperation, closing with an explicit
alignment section.

Ready-to-send file: `Jeremias_Likerman_CV_UNESCO_IGCP.pdf`.

Compile from this directory with:

```bash
xelatex main.tex
xelatex main.tex
```
