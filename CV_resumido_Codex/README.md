# CV resumido para carbonatos naturalmente fracturados

Archivo principal: `main.tex`.

Compilar con LuaLaTeX desde Texmaker. También debería funcionar con XeLaTeX si está instalado, pero esta versión fue verificada con LuaLaTeX. El archivo usa:

- `awesome-cv.cls` copiado localmente en este directorio.
- Fuentes del repositorio en `../fonts/`.

No requiere BibLaTeX/Biber: las publicaciones relevantes están cargadas manualmente para mantener el CV compacto.
